## Guidelines for my CSS

Define the scope of the var

```javascript
:root {
--black: #333
}

// Use the style
h1 {
color:var(--black)
}

// use the style as var in the media
@media only screen and(max-width:600px){
body {
  --black: magenta;
}
}
```
